# Sportsclubmanagementsystem
A C program for managing sports club records
I recently worked on a Sports Club Management System using C language as part of my first-semester learning journey in engineering. This project was my attempt to automate that process using basic programming concepts. Through it, I got hands-on experience with structures, functions, menu-driven programs, and file handling in C. The system allows users to add, edit, delete, and view player records, manage court reservations, and securely store data using files, with a simple password-protected view for added security.
